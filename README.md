# SimpleSoundtrackManager
The automated soundtrack system allows a user to seamlessly transition between audiotracks.
The primary focus of this application is to be used during roleplaying/tabletop games, or other live events where control over background audio is required.

## Track View
The track view is where you setup a session. By adding different tracks, customizing their start and loop point, as well as the transition length.

## Session View
The session view is where you control an active session. By clicking on a track it begins playing, using the selected tracks transition length to guide the fade in/out.
